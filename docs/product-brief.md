# Product brief

## 商品定義

Signal Chain Preflightは、PC・ゲーム機・ドック・変換アダプター・ケーブル・モニターからなる映像／給電経路を、購入前に区間単位で診断するブラウザ完結型ユーティリティである。単なる帯域計算ではなく、成立条件、最初のボトルネック、機能継続性、不明情報、最小変更案、購買仕様、判定証跡を返す。

## 対象ユーザー

- 高解像度・高リフレッシュレート環境を組むPC／ゲーム機ユーザー
- USB-Cドックや変換アダプターを購入前に比較するユーザー
- 問い合わせや導入設計で「どこまで通るか」を説明する販売・IT担当者

## JTBD

機器を買う前、または映像が期待どおり出ない時に、端子名や規格バージョンの宣伝文句ではなく、実際に必要な能力と経路上の不足を確認し、最小の変更で成立させたい。

## MVP

単一映像ストリーム、線形経路、DisplayPort HBR2/HBR3/UHBR、USB-C DP Alt Mode 2/4レーン、公開根拠が揃う範囲のHDMI、DSC/HDR/VRR継続性、簡易USB PD、URL hash共有を扱う。

## 非対象

USB4/Thunderboltトンネリング、MST、複数画面、HDCP、eARC、個別HDR形式、OS/ドライバー制約、信号品質や長さ保証、個別PDO交渉、実商品ランキング、販売価格、在庫、アフィリエイトを扱わない。

## 収益化の境界

診断エンジンは商品名・価格・アフィリエイト情報に依存しない。結果はカテゴリ別の購買要件までを生成し、将来の商品候補は独立したRecommendationモジュールが要件に適合する根拠と広告関係を明示して表示する。
