# Signal Chain Preflight v0.1.0-alpha.1

最初の公開αです。単一映像チェーンを、connectorやversion labelではなく区間ごとの明示能力で診断します。

## Included

- DisplayPort HBR2/HBR3/UHBR、USB-C DP Alt Mode、HDMI TMDS
- DSC/HDR/VRRの継続性、簡易USB PD
- 四状態判定、Unknown伝播、bottleneck、proof、fallback、購買要件
- URL hash共有、portable JSON report、公開Issue forms

## Alpha boundary

HDMI FRLの精密payload、USB4/Thunderbolt、MST/複数画面、HDCP/eARC、OS/driver、物理信号品質、製品型番保証は含みません。特にFRLを公開raw bandwidthだけで成立扱いしません。

完全な変更一覧と検証状態は `CHANGELOG.md` と `docs/release-checklist.md` を参照してください。
